# ADR 0046: Narrow Trusted Promotion Policy v2

- Status: Accepted
- Date: 2026-07-17

## Decision

Automatic Trusted Promotion is limited to fully qualified Claims and independently reused Concepts with a valid current Receipt. Concept independence uses distinct Work IDs. Automatic promotion is paused for Question, Tension, Hypothesis, Analogy, Anomaly, Intuition and Synthesis.

## Consequences

Repeated scans or URLs cannot promote exploratory objects. Manual promotion remains possible with an explicit reason.
