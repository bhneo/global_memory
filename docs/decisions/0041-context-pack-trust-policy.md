# ADR 0041: Context Pack trust policy

- Status: accepted
- Date: 2026-07-17

## Decision

Context Pack exposes the tier explicitly as `truth_layer`. Research and exploration may use Working with provenance and uncertainty visible; execution admits Working only for questions and tensions unless another profile is combined. Trusted and Canonical receive higher rank. Explicit status filters override defaults.

Retrieval never changes tier or treats a hit as confirmation.
