# ADR 0048: Read-only Semantic Drift Audit

- Status: Accepted
- Date: 2026-07-17

## Decision

Compare versions for uncertainty erasure, correlation-to-causation, secondary-to-primary, analogy-to-fact, unsupported synthesis additions and translation strengthening. Reports recommend action but never rewrite Trusted/Canonical; high severity enters Exception.

## Consequences

Model-assisted drift detection is advisory and auditable rather than an ungoverned writer.
