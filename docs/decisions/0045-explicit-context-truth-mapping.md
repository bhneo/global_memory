# ADR 0045: Explicit Context Truth Mapping

- Status: Accepted
- Date: 2026-07-17

## Decision

Context items expose tier, epistemic status, confidence, authority, coverage, entailment, unresolved contradictions and last consolidation. Source and Proposal use explicit layers; unknown legacy state maps to Unknown, never Canonical. Execution excludes unsafe exploratory states by default.

## Consequences

Research and Exploration remain broad but labeled. Execution cannot launder contested, hypothetical or unknown material into facts.
