# ADR 0027：Epistemic Verification Dimensions

- 状态：accepted
- 日期：2026-07-16

## 决策

分别记录 quote verification、extraction quality、source authority、evidence entailment 与 claim confidence。exact quote 只证明定位，不证明 extraction 完整、来源权威、证据完整蕴含或 claim 成立；任一维度降级都必须保留并影响 publication gate。

## 结果

系统避免把“找到了原句”误当作“知识已确认”。
