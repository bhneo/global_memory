# 主动真机因子评测

把机器人策略在对象位姿、相机视角和初始状态等结构化任务因子组合上的性能视为未知函数，用带不确定性估计的概率代理模型和信息增益准则依次选择真机试验，以在有限预算下估计性能分布并定位易失败区域。
