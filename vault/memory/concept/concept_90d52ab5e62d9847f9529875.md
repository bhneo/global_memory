# VLA 注意力泛化—动作泛化缺口

VLA 可以在未见场景或本体中形成面向操作对象与接触区域的可迁移注意力，却仍缺少把这些区域稳定映射为可执行动作的能力；因此视觉注意力泛化与动作级成功必须分开评价。
